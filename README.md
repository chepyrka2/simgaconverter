# simgaconverter
___
## ENGLISH
### Requirements
> python 3.12
> 
> pillow==11.0.0
### How to change language?
- Open lang.ini
- Change lang to **eng** or **rus**
### How to launch?
- open cmd or terminal from converter folder
- python coreapp.py
### Second variant
- launch the project in PyCharm
